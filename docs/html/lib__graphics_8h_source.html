14/* about this library: Graphics
15 *
16 * This provides functionality for accessing the VICKY's bitmaps, sprites, etc.
17 *
18 *** things this library needs to be able to do
19 * initialize/enter graphics mode
20 * exit graphics mode
21 * fill an entire screen of bitmap
22 * fill a rect
23 * draw a rect
24 * allocate a bitmap
25 * copy a bitmap
26 * load a bitmap from disk
27 * draw a circle
28 * fill an enclosed area
29 * draw a round rect
30 * draw a line
31 * get the value of a pixel, from specified x/y coords
32 * set the value of a pixel, from specified x/y coords
33 * copy a rect of pixel mem from one bitmap to another
34 * copy a rect of pixel mem from place to place within the same bitmap
35 *
36 * STRETCH GOALS
37 * load a graphical (proportional width or fixed width) font from disk or memory
38 * draw string using graphical font on screen, at specified x/y
39 * draw string using graphical font on screen, wrapping and fitting to specified rectangle
40 *
41 * SUPER STRETCH GOALS
42 * have a clipping system that prevents drawing to non-clipped parts of the screen
43 * have a layers system that prevents drawing to portions of layers that are under other layers
44 *
45 */

278// Generally speaking (the order does not really matter on this):
279// 1. Set the mode bits in the master control register (GRAPHICS + BITMAP), as well as the resolution.
280// 2. Load the color table for the bitmap into one of the graphics LUTs (start at $B4:2000 on the A2560U, $FEC8:2000 on the A2560K)
281// 3. Load the bitmap pixel data into video RAM (starts at $C0:0000 on the A2560U, $0080:0000 on the A2560K)
282// 4. Set the enable bit and LUT # in the bitmap control register for the layer you want
283// 5. Set the address of the pixel data in video RAM in the VRAM address pointer for the layer you want. This must be set relative to the beginning of VRAM, so if you're using the A2560U and the pixel data starts at $C2:0000, you'd store $02:0000 in the address register.
284// Paul Scott Robson â€” Today at 12:45 PM
285// TextInitialise:
286// clr.l d0
287// lea vicky3,a0 ; start register writes
288// move.l #$0000000C,(a0) ; graphics mode & bitmap on.
289// move.l d0,$4(a0) ; zero border size/scroll/colour
290// move.l d0,$8(a0) ; clear border colour
291// move.l d0,$C(a0)
292// move.l d0,$10(a0) ; cursor off
293// move.l d0,$14(a0) ; no line interrupts
294// move.l d0,$18(a0)
295//
296// move.l #$00000001,$100(a0) ; LUT 0, BMP 1 on
297// move.l d0,$104(a0) ; BMP 1 address
298// move.l d0,$108(a0) ; BMP 2 off
299//
300// move.l #$0000FF00,$2004(a0) ; colour 1 red
301// move.l #$00FF00FF,$2008(a0) ; colour 2 green
302// move.l #$00FFFFFF,$200C(a0) ; colour 3 yellow
303// move.l #$FF0000FF,$2010(a0) ; colour 4 blue
304// move.l #$FF00FFFF,$2014(a0) ; colour 5 magenta
305// move.l #$FFFF00FF,$2018(a0) ; colour 6 cyan
